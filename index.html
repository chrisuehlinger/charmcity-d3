What is D3?

  • Data Driven Documents
  • Built by Mike Bostock
  • It can draw using:
    • HTML elements (<div>'s, <span>'s, <ul>'s, <li>'s)
    • SVG elements (<g>'s, <circle>'s, <rect>'s, <text>'s)
    • Canvas (using the browser's Canvas API)

D3 Code


d3.select(".chart")
  .selectAll("div")
    .data(data)
  .enter().append("div")
    .text(function(d) { return d; });
